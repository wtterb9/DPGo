# DarkPawns Legacy Races

DarkPawns player race options included:

- Human
- Elven
- Dwarven
- Kenderkin
- Minotauran
- Rakshasan
- Ssauran

DPGo uses GoMUD's race/stat system at runtime, but this legacy list is preserved
to reflect original setting flavor and references in imported world content/help.

## Notes

- Runtime race mechanics, gear-slot behavior, and combat math follow GoMUD.
- Legacy race descriptions and references are preserved where possible through
  imported help text and world descriptions.
- Use `help races` for current in-engine race details.
