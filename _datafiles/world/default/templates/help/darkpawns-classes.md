# DarkPawns Legacy Classes

DarkPawns historically used a class-based progression model.  
DPGo keeps GoMUD's native skill/job progression, but preserves this legacy
class context for players familiar with the original world.

## Legacy Class List

- Magic User
- Cleric
- Thief
- Warrior
- Magus
- Avatar
- Assassin
- Paladin
- Ninja
- Psionic
- Ranger
- Mystic

## In DPGo

- Character growth is still driven by GoMUD skills, jobs, and training systems.
- Legacy classes are presented as lore/identity guidance, not as hard engine roles.
- Use `help jobs` and `jobs` to understand your current GoMUD role progression.
